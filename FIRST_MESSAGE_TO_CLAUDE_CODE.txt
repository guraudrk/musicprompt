6. Gemini 구현은 다음 원칙을 지켜라.

   - API 키는 `GEMINI_API_KEY` 환경변수에서 서버 측으로만 읽는다.
   - 브라우저 번들에 키를 포함하지 않는다.
   - 현재 공식 Google GenAI SDK와 API 문서를 먼저 확인한다.
   - 구조화 출력과 Zod 검증을 사용한다.
   - 이론 엔진이 만든 구조 데이터를 Gemini에 전달한다.
   - Gemini는 최종 문장 정리와 Provider별 Prompt Package 생성을 담당한다.
   - 생성 후 규칙 기반 검증과 별도 평가를 거친다.
   - 스키마 또는 차단 오류가 있을 때만 자동 수정 1회를 허용한다.
   - 사용자의 원문 가사와 잠긴 필드를 보존한다.
